# Number-Guessing-Game

## Overview
The **Number-Guessing-Game** is a simple Python program where the player guesses a number chosen randomly by the computer. You have a total of 7 attempts to guess the correct number.

## How It Works
- The computer randomly selects a number.
- The player enters guesses one by one.
- After each guess, the program provides feedback:
  - If the guess is too high or too low.
  - If the guess is correct.
- The game ends when:
  - The player guesses the correct number.
  - The player uses all 7 attempts without guessing correctly.

## Technologies Used
- Python (standard library, including the `random` module).

# Word-Guessing-Game

## Overview
The **Word-Guessing-Game** is another fun Python-based game where players try to guess a randomly selected word from a predefined list of words.

## How It Works
- The computer selects a random word from a list.
- The player guesses one letter at a time.
- After each guess, the program:
  - Reveals the letters correctly guessed.
  - Tracks and displays the remaining attempts (10 guesses total).
- The game ends when:
  - The player successfully guesses all the letters in the word.
  - The player runs out of guesses.

## Technologies Used
- Python (standard library, including the `random` module).
